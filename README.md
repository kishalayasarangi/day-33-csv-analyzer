# Day 33 — CSV Data Analyzer

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A Python CSV data analyzer that reads any CSV file
and generates stats, charts and insights automatically.

## How to run
```bash
pip3 install pandas matplotlib
python3 analyzer.py
```

## Features
- Auto detects numeric and text columns
- Mean, median, mode, std dev for numeric columns
- Value counts for categorical columns
- Generates charts automatically
- Works on any CSV file
